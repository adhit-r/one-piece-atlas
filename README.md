# One Piece World Atlas

Interactive globe & dataset of islands, arcs, and episode mappings for One Piece.

## Run locally

- Install Bun: https://bun.sh
- Install deps: `bun install`
- Dev: `bun run build:dev && bun run dev`
- Build: `bun run build`
- Tests: `bun test`
